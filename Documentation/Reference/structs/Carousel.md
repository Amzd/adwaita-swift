# `Carousel`

A carousel view.

## Properties
### `elements`

The elements.

### `content`

The content.

### `allowLongSwipes`

Whether long swipes are allowed.

### `view`

The view.

## Methods
### `init(_:content:)`

Initialize `Carousel`.
- Parameters:
  - elements: The elements.
  - content: The view for an element.

### `longSwipes(_:)`

Set whether long swipes are allowed or not.
- Parameter longSwipes: Whether long swipes are allowed.
- Returns: The carousel.
